Testing

  • Adaptivity Tests: DUT shoud avoid transmitting in the presence of interference in ISM band, e.g. 2.4GHz
    • Ref
      • ED by NXP
      • ETSI spec: 2.4GHz spec, 5GHz spec
    • COT (Chan Occupation Time), Timing
    • ED:
      • One way to test is to measure tput using iperf (e.g. thread): Let DUT transmits to REF while increasing the interference to DUT. As the interference exceeds the ED threadhold, the Rx tput measured by iperf on REF should drop or disappear.
      • Another way to test is to measure spectrum (e.g. WLAN): Let DUT transmits to REF while increasing the interference to DUT. As the interference exceeds the ED threadhold, the DUT should vacate the portion of the spectrum occupied by the interference.
  • Emission tests: OOB, Bandedge, Spectrum
    • Ref
      • ETSI has a series of spec called "Harmonised Standard for access to radio spectrum" that covers various bands (2.4GHz, 5GHz, 6GHz) and various devices (BS/AP, UE/ST, WLAN, LTE/NR).
