      • center freq, span, RBW/VBW, detection mode (peak, average, RMS), sweep mode (continuous/single)
      • sweep time: typically automatically determined by the settings above
      • trace mode: clear/write, max/min hold (can be cleared by reset or trace clear cmd), average (needs to choose between RMS average and log or dB average, needs to set average count: 10 - 50 for moderately stable signal with some noise. Larger count yields more smooth curve but takes longer measurement time).
  • Overview
    • Rx requirements
      • Sensivity = req_SNR + NF + 10*log10(kTB) (-174 dBm/Hz + 10*log10(B))
      • Dyn Range = MaxInputSig - MDS (Min Detectable Signal above noise) = P1dB (1dB compression pt) - MDS
      • RxEVM = a metric used at high input. SNR_dB = -20*log10(RxEVM*100)
      • Inpairment: IQ imbalance/DC offset, thermal, inband LO phase noise, passband ripple, non-linearity ()
      • Interference: blocker, adj channel, intermodulation (two-tone IIP3/IIP2)
    • Tx requirements
      • Power: Tx Power range/ctrl, gain control accuracy, Time mask (pre-burst level, overshoot/undershoot)
      • Inband Signal Quality: EVM
      • Outband Signal Quality: ACLR, Spectral Mask
    • Metric cascade
      • NF (Friis) = 10*log10(Ftotal), where Ftotal = F1 + (F2-1)/G1 + (F3-1)/(G1*G2) + ...
      • 1/IIP3_total = 1/IIP3_1 + G1/IIP3_2 + G1*G2/IIP3_3 + ...
      • EVM_total2 = EVM12 + EVM22 + ...
        • Phase noise (smearing in the consterlation diagram)
        • Filter response (not as critical for OFDM)
        • DC offset (not as critical for OFDM)
        • IQ imbalance, Single Sideband Rejection (SSB) aka Image Rejection (3dB worse effect for OFDM)
        • PA non-linearity: AM-AM, AM-PM
    • Rx arch: Zero-IF or direct conversion, Low-IF, Polar modulator
