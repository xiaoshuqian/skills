  • Remember
    • A URL has three parts: Protocol (http, ftp, etc), Domain, path (default to index.html)
    • Reques/Response cycle: DNS lookup, Req/Res, Display
    • HTML is based on DOM (Doc Obj Model or tree like a struct: root elem html -> head + body
    • Elem, tag, attribute, value
    • In HTML5, it is recommended to avoid using />: omit the self-closing slash for void elements like link, br, img, and meta
