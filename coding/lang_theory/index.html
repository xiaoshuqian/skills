    • 5 aspects: syntax (EBNF or regex diagram), semantics, idioms, lib, tools
    • Roadmap
      • Data types: primitive data types: int, float, bool, str, None/null; operations: +-*/%, concatenate; precedence/associativity, flat/compound: arrays, mutability
      • variables, lexical/global scopes, namespace, functions, class, copy/pass by value/ref, function/obj factory (a function that returns a function or obj), closure (a function plus its own exec env or data atrributes)
      • Control structures: sequential, conditional, loops
      • Hello worlds, compiling, execution, REPL
      • Packaging
      • File I/O, Lib, comments
