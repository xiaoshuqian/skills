  • Keyboard shortcuts
    • Ref
      • support.apple.com/en-us/102650
      • apple menu -> system settings -> keyboard -> keyboard shortcuts
      • Help menu (on menu bar of MAC or any APP): search "keyboard shortcuts"
    • Function keys:
      • F11: goto Desktop
    • cmd + ctrl
      • Q: sleep or Quit
      • F: toggle Full screen
      • M: merge find or terminal windows
      • R: Refresh website
    • cmd-shift
      • 5,4 (or 3): print an area of the screen (or the window) to the file (or to the clipboard if ctrl is also pressed). Also, see How to annotate a screen capture
    • cmd
      • + or -: zoom in or out website or app window
      • _ (space): spotlight search
      • R: open the folder, used with spotlight
