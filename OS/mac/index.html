MAC

  • Keyboard shortcuts
    • Ref
      • support.apple.com/en-us/102650
      • apple menu -> system settings -> keyboard -> keyboard shortcuts
      • Help menu (on menu bar of MAC or any APP): search "keyboard shortcuts"
    • cmd^ (cmd + ctrl)
      • Q: sleep or Quit
      • F: toggle Full screen
      • M: merge find or terminal windows
      • R: Refresh website
    • cmd
      • + or -: zoom in or out website or app window
      • _ (space): spotlight search
